## Overview

Cards are components that usually contain some content and typically actions about a context.

Normally, a `Card` consists of:

1. `Card-header`: Usually contains a title to describe the **subject** of a card. Semantically, it should be located at the **very top** of the card. Can be used interchangably with `.Card-media`.
2. `Card-media`: Displays the **thumbnail** of the card; be it video or image, that describes the content. Usually located at the top of a card **after** `.Card-header`.
3. `Card-body`: Contains the **text** or **paragraph** that describes the content of a card.
4. `Card-footer`: Usually contains **actions** (e.g button or links) where users can interact with.
