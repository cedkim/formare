At the most basic level, apply the `.u-Flex` class to transform an element into a flex container.
All direct children inside this container will also be converted into flex items.

The `.u-Flex` class will apply `display: flex` property. Use `.u-InlineFlex` class instead to apply
`display: inline-flex`.

```html preview=true
<div class="DummyBlock u-Flex u-MarginBottom">
  <div class="DummyBlock u-MarginRight">Flex Item</div>
  <div class="DummyBlock u-MarginRight">Flex Item</div>
  <div class="DummyBlock">Flex Item</div>
</div>

<div class="DummyBlock u-InlineFlex DummyBlock">
  <div class="DummyBlock u-MarginRight">Flex Item</div>
  <div class="DummyBlock">Flex Item</div>
</div>
```

## Flex Direction

There are two types of flex direction utility classes that you can use:

1. `.u-Flex-row`
2. `.u-Flex-column`

Use `.u-Flex-row` instead to create a flex container with `flex-direction: row` property to **horizontally** align flex items.

```html preview=true
<div class="DummyBlock u-Flex u-Flex-row">
  <div class="DummyBlock u-MarginRight">Flex Item</div>
  <div class="DummyBlock u-MarginRight">Flex Item</div>
  <div class="DummyBlock u-MarginRight">Flex Item</div>
</div>
```

Use `.u-Flex-column` instead to create a flex container with `flex-direction: column` property to **vertically** align flex items.

```html preview=true
<div class="DummyBlock u-Flex u-Flex-column u-Flex:">
  <div class="DummyBlock u-MarginBottom">Flex Item</div>
  <div class="DummyBlock u-MarginBottom">Flex Item</div>
  <div class="DummyBlock">Flex Item</div>
</div>
```
