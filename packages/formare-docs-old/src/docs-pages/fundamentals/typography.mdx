## Headings

We use the **Minor Third** type scale from [typescale.com](typescale.com) to construct
our headings in which you can customize to your liking.

<!--
<Examples.HeadingsTable /> -->

| Tag    | Class Variant | Result             |
| ------ | ------------- | ------------------ |
| `<h1>` | `.Heading-1`  | <h1>Heading 1</h1> |
| `<h2>` | `.Heading-2`  | <h2>Heading 2</h2> |
| `<h3>` | `.Heading-3`  | <h3>Heading 3</h3> |
| `<h4>` | `.Heading-4`  | <h4>Heading 4</h4> |
| `<h5>` | `.Heading-5`  | <h5>Heading 5</h5> |
| `<h6>` | `.Heading-6`  | <h6>Heading 6</h6> |

Combine headings with `.Title` and `.Subtitle` classes.

```html preview=true
<h1 class="Title">I'm a bolded title!</h1>
<h3 class="Subtitle u-Color-gray">Subtitle text</h3>
```
