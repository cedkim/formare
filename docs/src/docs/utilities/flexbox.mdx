Use `.u-Flex` class to apply `display: flex` to transform an element into a flexbox container and turn its direct
children into flex items. Use `.u-InlineFlex` to apply `inline-flex` instead of the standard `flex`.

#### Flex Direction

There are two types of flex direction utility classes that you can use:

1. `.u-Flex-row`
2. `.u-Flex-column`

Use `.u-Flex-row` instead to create a flex container with `flex-direction: row` property to **horizontally** align flex items.

<Preview lang="html" layout="horizontal">
  <PreviewDisplay>
    <div className="u-Flex u-Flex-row">
      <div className="DummyBlock u-MarginRight">Flex Item</div>
      <div className="DummyBlock u-MarginRight">Flex Item</div>
      <div className="DummyBlock">Flex Item</div>
    </div>
  </PreviewDisplay>
  <PreviewCode>
    {`
    <div className="u-Flex u-Flex-row">
      <div className="DummyBlock">Flex Item</div>
      <div className="DummyBlock">Flex Item</div>
      <div className="DummyBlock">Flex Item</div>
    </div>
    `}
  </PreviewCode>
</Preview>

Use `.u-Flex-column` instead to create a flex container with `flex-direction: column` property to **vertically** align flex items.

<Preview lang="html" layout="horizontal">
  <PreviewDisplay>
    <div className="u-Flex u-Flex-column">
      <div className="DummyBlock u-MarginBottom">Flex Item </div>
      <div className="DummyBlock u-MarginBottom">Flex Item </div>
      <div className="DummyBlock">Flex Item</div>
    </div>
  </PreviewDisplay>
  <PreviewCode>
    {`<div className="u-Flex u-Flex-column">
      <div className="DummyBlock">Flex Item</div>
      <div className="DummyBlock">Flex Item</div>
      <div className="DummyBlock">Flex Item</div>
    </div>
    `}
  </PreviewCode>
</Preview>
